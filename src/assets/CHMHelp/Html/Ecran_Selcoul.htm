Color Selection

You can set background and text colors using this screen. Simply click on a color to select it. The current color is surrounded by a black flashing frame. You can also use the arrow keys to select a color.

Shade of the Background Color can be adjusted more finely using the slider. You can click directly on the gradient colors to select a shade. The right side of this screen shows colors used by Windows. These colors can be changed from Control Panel of Windows.

The [Personal Colors ...] button allows you to define additional personal colors.


Click the [Ok] button to confirm the selected colors and parameter, or the [Cancel] button to keep the current graphic parameters.
