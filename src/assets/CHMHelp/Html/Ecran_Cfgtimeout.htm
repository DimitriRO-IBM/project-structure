Internet Options

Internet Time-Out

SACR uses IBM Network to download SRM Data. Download SRM information can be a quite long process. So, to avoid Internet Time-Out Errors, the program automatically sets the Internet Time-Out value to 2 minutes. But you can change this value if you want (up to 15 minutes).

Select the Time-Out value (in minutes) and press the button [Ok] to save your selection.


HTTPS Download Method

Metrics are downloaded from SRM and Cognos using HTTPS Protocol.
You can select one of these two methods to perform SRM and Cognos Metrics downloading :
  • HTTPS Protocol
  • Using external Curl Module (Default)
