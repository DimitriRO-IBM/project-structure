View Details

This window displays complete information related with a Server.
The top of the window gives general information :

- Customer Name
- Server Name
- Subclient ID 1
- Subclient ID 2
- Type ("Server" or "LPAR")
- Operating System
- OS Level
- Box Name
- Model Number

Metric Selection
You can select the Metric to be displayed using the associated buttons :
  • CPU
  • Memory
  • Disk

Memory Calculed Metrics

For AIX (Lpars and Servers) and LINUX (Servers), the program displays a calculated Metric : [Memory Available for Applications (Mo)].
These are the formulas used to calculate this Metric :

AIX :
[mem_pages_avail_applications] = MAX((free pages - minfree),0) + MAX((file pages - minperm),0)
[Memory Available for Applications (Mo)] = ( [mem_pages_avail_applications] x 4096 ) / (1024 x 1024)

LINUX Server :
[memory_effective_used_percent] = (memory free + memory buffer + memory cache)/(installed memory)
[Memory Available for Applications (Mo)] = [memory_size] - ( [memory_effective_used_percent] x [memory_size] / 100 )


Thresholds saved in memory are displayed according the selected Metric.

For each Metric, the program displays on a table the SRM Fields and the corresponding values for the 4 last months.


Commands :

Button [Volume Groups]
Click on this button to display the list of Volume Groups of for the selected server. You will open Volume Groups.

Button [Filesystems]
Click on this button to display the list of Filesystems of for the selected server. You will open Filesystems.

Button [Monthly View]
Click on this button to display Daily data and Hourly data. You will open Monthly View.

Button [Forecast View]
Click on this button to display Forecast Values. You will open Forecast View.

Button [Copy Infos]
Click on this button to copy general information of the Server (Customer, Server ...). You can then paste the information on another program using [Cntrl+V] shortcut.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. You will first obtain Copy Table window in order to select the copy format. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain Graphic Size Selection.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing Tables for the three Metrics. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
