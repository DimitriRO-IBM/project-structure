RVTools - Disks and Partitions

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.

OS Selection
You can filter the VM list by Operating System :
  • All
  • Linux
  • Windows
  • Other

Searching
To search a VM, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Item line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Items.


2 - List of VMs

This list displays the VMs for the selected Customer, VCenter and Date. Click on the name of each column to sort it. You can obtain more details for each VM with a double click on a VM line.
Each time you click on a VM, SACR will display two secondary lists :
  • List of Disks
  • List of Partitions


Commands :

Button [Red FS /LU]
Click on this button to display Red Filesystems or Red Logical Units.
You will obtain the Red File System Window.

Button [VM Details]
Click on this button to display all the VM details.
You will obtain the VM Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of Disk and the list of Partitions for the selected VM.
The page will be displayed using the selected HTML Browser.

Button [Copy Tables]
Click on this button to copy the list of the VMs or the list of Partitions. The program will ask you to select the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
