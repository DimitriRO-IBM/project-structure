Collect Quality Formulas


Daily Calculation

This is the Formula used to calculate the Daily Collect Quality Percentage :

pct_collected_per_server = (Number of collected per datatype / Number of expected per datatype) * 100
Example :
On SRM, a server have 5 datatypes. SRM collected 3 datatypes:

%Collected = (3 datatypes Ok / 5 datatypes expected) * 100 = 60% for this day


Monthly Calculation

This is the Formula used to calculate the Monthly Collect Quality Percentage :

pct_collected_per_server = (Number of collected per datatype * Number of days collected of that month) / (Number of expected per datatype * total Number of days of the month) * 100
Example :
On SRM, a server have 5 datatypes. SRM collected 3 datatypes Ok during 24 days and 4 datatype Ok during 6 days :

%Collected = [(3 datatypes ok * 24 jours) + (4 datatypes ok * 6 jours)] / (5 datatypes expected * 30 days) * 100 = 64% for the whole month
