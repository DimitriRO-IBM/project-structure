RVTools - ESX Locations

This window is designed to define ESX Locations.

Step 1
Select the method to define ESX Locations :
  • By Clusters Names
  • By ESX Names

Step 2
Select in the list Clusters (or ESX) to be updated with the Location Name.

Step 3
Give the Name of the Location, and then click the button [Update].

Step 4
Repeat these operations for others Clusters (or ESX) and Locations.

Step 5
Click the [Ok] button to finish and update ESX Locations in the Database.
