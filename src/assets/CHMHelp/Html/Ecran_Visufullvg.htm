Red Volume Groups / Red Filesystems

This window displays the list of all Volume Groups / Filesystems for the selected Customer in a Red Status.
The top of the window gives general information :

- Customer Name
- Subclient ID 1
- Subclient ID 2
- Selected Month
- Operating System

Commands :

Status Selection
You can display only items in a Red Status (default), or all items.

Button [View Details]
Click on this button to open the View Details window for the selected Server on the list. You obtain the same result with a double click or pressing the [Enter] key.

Button [Volume Groups]
Click on this button to display the list of Volume Groups of for the selected server. You will open Volume Groups.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graphic. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
