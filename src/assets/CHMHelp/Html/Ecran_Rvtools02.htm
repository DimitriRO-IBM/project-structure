RVTools - Excel Import File

Import of a RVTools Excel File is done in 4 steps :

Step 1 - Type, or select with the list, the Customer Name and the VCenter Name
Type or select the Customer Name and the VCenter Name using the two combo lists.

Step 2 - Excel File Selection
Click on the button [Browse ...] to select and open a RVTools Excel File.
Imported Data :
  • Tab [List of VMs]
    List of VMs are the fusion of these Excel Tabs :

    - [tabvInfo]
    - [tabvCPU]
    - [tabvMemory]
    - [tabvTools]
    - [tabvHealth]

  • Tab [Disk Data]
    This Tab contains these Excel Tab : [tabvDisk Tab]

  • Tab [Disk Partitions]
    • This Tab contains these Excel Tab : [tabvPartition]

  • Tab [Cluters]
    • This Tab contains these Excel Tab : [tabvCluster]

  • Tab [ESX]
    • This Tab contains these Excel Tab : [tabvHost]

  • Tab [Datastore]
    • This Tab contains these Excel Tab : [tabvDatastore]

  • Tab [Network]
    • This Tab contains these Excel Tab : [tabvNetwork]

Buttons [Modify Tables]
If necessary, you can modify now the name of the Customer or the name of the VCenter using these two buttons.
Just give the name of the Customer or the name of the VCenter on the combo box, and click on the associated |Modify Tables] button.

Step 3 - Define Locations
RVtools Excel Files do not have the "ESX Location" information. So you have to input this information manually using the button [Define ESX Locations] (optional).
You will obtain the ESX Locations Window.

Step 4 - Save Data in RVTools Database
Click on the button [Save Data in Database] to finish import.

Button [Quit]
Click on this button to return to the previous screen.
