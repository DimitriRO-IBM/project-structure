1 - Aix and Linux Thresholds

This window is designed to input Aix / Linux Default Thresholds Values as following :




1 - Highest Values

CPU

Thresholds are defined by 4 values :
  • Lower Server CPU Usage (%)
  • Upper Server CPU Usage (%)
  • Lower LPAR CPU Usage (% Entitled Capacity)
  • Upper LPAR CPU Usage (% Virtual Processor Count)
Memory

Thresholds are defined by 8 values :
  • AIX : Lower Memory Swap Usage (%)
  • AIX : Upper Memory Swap Usage (%)
  • LINUX : Lower Memory Usage (%)
  • LINUX : Upper Memory Usage (%)
  • Lower Page Ins
  • Upper Page Ins
  • Lower Page Out
  • Upper Page Out
Disk

Thresholds are defined by 2 values :
  • Free Size (Mb)
  • Disk Used (%)


2 - Lowest Values

CPU

  • Up To Server CPU Usage (%)
  • Up To LPAR CPU Usage (%)
Memory

  • Up To Used Melory (%)
