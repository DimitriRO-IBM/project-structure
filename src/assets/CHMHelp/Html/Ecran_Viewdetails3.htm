Forecast View

This window displays Forecast Values for CPU, Memory and Disk Metrics for the selected Server.
The top of the window gives general information :

- The Month to be displayed
- The Customer Name
- The Server Name
- The Type ("Server" or "LPAR")
- The corresponding CPU Thresholds

Commands :

Button [Display URL]
Click on this button to show SOAP URL in a single line.

Button [Test SOAP Query]
Click on this button to launch the SOAP Query in the Web Browser.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph of the selected Tab on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing Tables for the Daily View and the Hourly View. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
