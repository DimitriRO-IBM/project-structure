Shared Pool - Details by LPAR

This window displays Processor Counts on every LPAR for :

- The selected Customer
- The selected Box
- The selected Day
- The selected Hour


Commands :

Button [Display URL]
Click on this button to show SOAP URL in a single line.

Button [Test Daily Query]
Click on this button to launch the Daily SOAP Query in the Web Browser.

Button [Test Query]
Click on this button to launch the SOAP Query in the Web Browser.

Button [View Details]
Click on this button to open the View Details window for the selected LPAR on the list. You obtain the same result with a double click or pressing the [Enter] key.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
