Default Thresholds for VG, LU and FS

Select first the OS :
  • Aix / Linux
  • Windows

1 - Aix and Linux Thresholds

This window is designed to input Aix / Linux Default Thresholds Values as following :




1 - Volume Group (VG)

Thresholds are defined by 2 values :
  • Lower Value (%)
  • Upper Value (%)


2 - Filesystem (FS)

Thresholds are defined by 8 values :
  • Filesystem Low Size [S1] (Giga Byte)
  • Filesystem High Size [S2] (Giga Byte)

  • Lower Threshold for FS < [S1] (%)
  • Upper Threshold for FS < [S1] (%)

  • Lower Threshold for [S1] < FS > [S2] (%)
  • Upper Threshold for [S1] < FS > [S2] (%)

  • Lower Threshold for FS > [S2] (%)
  • Upper Threshold for FS > [S2] (%)


2 - Solaris, HP-UX Thresholds

This window is designed to input Solaris and HP-UX Default Thresholds Values as following :



Filesystem Thresholds are defined by 8 values :
  • Filesystem Low Size [S1] (Giga Byte)
  • Filesystem High Size [S2] (Giga Byte)

  • Lower Threshold for FS < [S1] (%)
  • Upper Threshold for FS < [S1] (%)

  • Lower Threshold for [S1] < FS > [S2] (%)
  • Upper Threshold for [S1] < FS > [S2] (%)

  • Lower Threshold for FS > [S2] (%)
  • Upper Threshold for FS > [S2] (%)
