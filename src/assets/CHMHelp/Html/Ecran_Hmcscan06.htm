Shared Pool Names / VIO Disks

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The HMC Name
  • The Date
Each time you select a new Customer, SACR will update automatically the "HMC" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the HMC list contains more than 2 HMC, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom HMC Selection.

List of Shared Pool Names / VIOs
If you double-click on a Box, or if you press the [Enter] key, you will open the Details Window.

Metric Selection (VIOs)
You can select with this list the values displayed on the graph :
  • Number of VIOs by Box
  • Number of Disks by Box
  • Total Disk Size by Box

Commands :

Button [Box Details]
Click on this button to view details of the selected Box.
You will obtain the Box Details Window.

Button [VIO Details]
Click on this button to view details of the selected VIO.
You will obtain the LPAR Details Window.

Button [List of LPARs]
Click on this button to view the list of LPARs for the selected Shared Pool.
You will obtain the Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of items.
The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the list on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the displayed graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
