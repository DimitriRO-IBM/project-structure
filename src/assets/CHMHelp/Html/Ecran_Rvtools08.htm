RVTools - Database Maintenance

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.


This window displays 3 lists :
  • The list of the Tables
  • The list of the Fields
  • The Contents of the selected Table
Each time you select a Table, SACR will display the list of Fields and the Contents of the Table.

Commands :

Button [Rename this Customer]
Click on this button if you need to rename the selected Customer.

Button [Rename this VCenter]
Click on this button if you need to rename the selected VCenter.

Button [Modify this Date]
Click on this button if you need to modify the selected Date.

Button [Delete this Selection]
Click on this button if you need to delete all data saved for the selected Customer, VCenter and Date.

Button [Import]
RVTools Data must be imported from RVTools Excel Files.
Click on this button to import a RVTools Excel File to the SACR Database.
You will obtain the Excel File Import Window.

Button [SQL Query]
You can display or modify the SQL Query used to fill the Contents list.
A new window will allow you to modify the current SQL Query using Access SQL commands.
Exemple :

SELECT * FROM VM
WHERE dateimp=20180419
AND customer="SCHNEIDER"
AND vcenter="VCENTER 1"
AND oslevel LIKE "*32*"

Button [Copy Table]
Click on this button to copy the list of Contains. The program will ask you to select the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
