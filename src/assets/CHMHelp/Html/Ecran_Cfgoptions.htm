Options

1 - HTML Viewer

All Reports are generated with HTML files. HTML files can be opened with various programs like :
  • Internet Explorer
  • Firefox
  • Google Chrome
By default, Windows associate HTML files with Internet Explorer (IE), but you can select another program to display such files.

Button [Selection]
Click on this button the select the HTML Viewer. A Popup Menu will be displayed :
  • 1 - Browse for a Specific Program
    Select this option to select a specific program. This selection will not affect the Windows association with HTML files (extension .htm or .html). It's only used by SACR.

  • 2 - Windows Default : XXXXX
    Select this option to use the Windows Default Browser.

  • 3 - Windows Internet Explorer (IE)
    Select this option to define Internet Explorer as HTML Files Viewer.

  • 4 - Microsoft Edge
    Select this option to define Edge as HTML Files Viewer.

HTML Background Color

You can select the Background Color of the HTML Reports generated by SACR.
Click on the color button to select a Background Color. You will obtain the HTML Color Selection Window.

Button [Default Color]
Click on this button to select the default HTML background color of the Reports.


2 - Microsoft Word

Reports are created using Microsoft Word. The program needs to know the path to find Word exe file "WINWORD.EXE".

Button [Browse ...]
Click on this button to select the path of WINWORD.EXE. Generally, WINWORD.EXE is located in the following directory :
C:\Program Files (x86)\Microsoft Office\...

If necessary, you can change the Word language : English or French.


3 - Reports Directory

Reports are done using HTML files. These files can be stored on a specific directory.

Button [Directory Selection ...]
Click on this button to select a specific directory. You will obtain the standard Windows Directory selector.

Button [Default Directory]
Click on this button to select default directory for the Reports :

C:\Capacity Reporting\Reports


4 - Saving the Options

Button [Ok]
Click on this button to save the selected options.

Button [Cancel]
Click on this button to quit without changes.
