End of Support Options

OS Versions are saved on a Configuration File named "OSVersions.ini".
This file is located on the directory of SACR program.
You can select one of these two Options :
  • 1 : Use File from SACR Server (default)

    Button [Reload from SACR]
    Click on this button if you want to download the OS Configuration file on your computer.

  • 2 : Use Local Configuration File
Press the button [Ok] to save your selection.
