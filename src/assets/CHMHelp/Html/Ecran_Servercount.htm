Infrastructure View / Boxes Consumption View

This window displays two kinds of values downloaded for the 13 last months :
  • Infrastructure View
    In this case, you can select one of the following Metrics :

    - Number of LPARs/Servers
    - Number of LPARs per Box
    - Number of Processors
    - Number of Virtual Processors
    - Memory Size in GB
    - Disk Size in TB

  • Boxes Consumption View
    In this case, select a Box on the list. The program will display, for the 13 last months :
    AIX
    - Number of LPARs
    - Number of CPU (Field config_cpu_count)
    - Number of CPU Max ( config_cpu_count X cpu_pct_max / 100 )
    - Number of CPU Used (Field config_cpu_count_used)

    AIX VIOs
    You can display VIOs information. Select in the list one of the two following options :
    - VIO View for the selected Box
    - VIO View, All Boxes

    For each VIO, the programm will display 3 values :
    - Processor Count
    - Entitled Capacity
    - Number of Virtual Processors

    Linux LPARs and Windows VM
    - Number of VM
    - % CPU Max (Field vc.cpu_max)

    VMWare ESX
    - % CPU Max (Field vc.cpu_max)
    - % Box Busy (Field box_percent_busy)

    VMWare Cluster
    - Number of ESX
    - % CPU Max (Field clusterutil.cpu_pct)
    - CPU Count (Field clusterutil.cpu_physical_count)

Commands :

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain Graphic Size Selection.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.
