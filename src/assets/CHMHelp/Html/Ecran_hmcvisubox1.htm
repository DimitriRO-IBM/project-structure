HMCScan - Box Details

This window displays all the Box Parameters.

Commands :

Button [VIO Details]
Click on this button to display details for the selected VIO.
You will obtain the LPAR Details Window.

Button [LPAR Details]
Click on this button to display details for the selected LPAR.
You will obtain the LPAR Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Box Details.
The page will be displayed using the selected HTML Browser.

Button [Box IO Slots]
Click on this button to display the list of IO Slots.
You will obtain the List of IO Slots Window.

Button [Quit]
Click on this button to return to the previous screen.
