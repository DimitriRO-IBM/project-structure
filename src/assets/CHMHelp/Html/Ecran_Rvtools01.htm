RVTools - VCenter Inventory

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.

Button [Import Excel File]
RVTools Data must be imported from RVTools Excel Files.
Click on this button to import a RVTools Excel File to the SACR Database.
You will obtain the Excel File Import Window.

Searching
To search an Item (Cluster, VM, ESX, Datastore ...), type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Item line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Items.


2 - Tab [VCenter View]

This Tab displays the number of Items for the selected Customer, the selected VCenter and the selected Date.
You can also select the VM Status :
  • All VMs (ON and OFF)
  • Only ON VMs
  • Only OFF VMs

Button [View Details]
You can display the list of Items concerned by the selection by pressing the [Enter] key or using this button.
You will obtain the Details Window.
Note : A double click on a line will open the same window.

Button [VCenter Architecture]
Click on this button to display the description of the VCenter Architecture.

Button [Define ESX Locations]
RVTools Excel Files do not have the "ESX Location" information. So you have to input this information manually using this button. You will obtain the ESX Locations Window.

Graphic Commands
These are the Graphic Commands.


3 - Tab [End of Support]

This Tab displays a table and some graphs.
  • The table shows, for every OS, the version number, the date of the End of Support and the number of Assets (ESX or VMs). SACR counts the number of Assets with the OS version "Ok", "To be Upgraded", "Not Supported" and "Not Applicable".
    The last line of the table displays the different totals.

  • The first graph is the "TOTAL OF ASSETS" graph.
    The following graphs are done for every OS.

End of Support Date Selection
You can display End of Support Counts using one of the 2 following dates :
  • Full Support Date
  • End Of Life Support Date
The 2 dates are displayed on the table in this way :
Full Support Date | End Of Life Date

Exemple :
01-2015 | 01-2020

Button [Options]
Click on this button to select the OS Configuration File to be used.
You will obtain the End of Support Options Window.

Button [End of Support Forecast]
Click on this button to display the End of Support Forecast Window.

Button [Details]
You can select / deselect one or more lines on different tables using the [Ctrl]+Clic command. Then, you can display the list of Assets concerned by the selection by pressing the [Enter] key or using the button [Details].
You will obtain the Details Window.
Note : A double click on a line will open the same window.


4 - Tab [Assets Inventory]

This Tab displays the list of Assets (ESX and VMs). You can sort the list clicking on the head of the columns.

Details
You can obtain more informtion with a double click on a line. You will obtain :
  • ESX Details if you select an ESX
  • VM Details if you select a VM

5 - Tab [Cluster View]

This Tab displays a Tree View with Clusters and ESX.
You can obtain more details with the following buttons :
  • [Cluster : List of ESX]
  • [Cluster : List of VM]
  • [Cluster : Details]
  • [ESX : List of VM]
  • [ESX : Details]
6 - Tab [ESX View]

This Tab displays a Tree View with ESX and VMs.
You can obtain more details with the following buttons :
  • [ESX : List of VM]
  • [ESX : Details]
  • [VM : Details]
7 - Tab [Datastore View]

This Tab displays a Tree View with Datastores and ESX.
You can obtain more details with the following buttons :
  • [Datastore : List of ESX]
  • [Datastore : List of VM]
  • [Datastore : Details]
  • [VM : Details]
8 - Tab [Location View]

This Tab displays a Tree View with Locations and ESX.
You can obtain more details with the following buttons :
  • [Location : List of ESX]
  • [ESX : List of VM]
  • [ESX : Details]

Commands :

Button
Click this button to collapse all Items on a Tree View.

Button
Click this button to expand all Items on a Tree View.

Button [View HTML]
Click on this button to generate a page on HTML format. You can select items to be displayed.
The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the displayed table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the displayed graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [Export CSV]
Click on this button to export Assets Inventory to a CSV file.
You will obtain Export to CSV Files window.

Button [Quit]
Click on this button to return to the previous screen.
