RVTools - Cluster Details

This window displays all the Cluster Parameters.

What is a Cluster ?

A Cluster is a group of ESX. When you add an ESX to a Cluster, the ESX resources
become part of the Cluster's resources. The Cluster manages the resources of all
ESX within it.

Clusters enable the Vphere High Availability (HA) end VSphere Distributed Resource
Scheduler (DRS) solutions.

See also : VCenter Architecture


Commands :

Button [Failover Info]
This button displays how are calculated values of CPU and Memory on Failover or Maintenance modes.

Button [Edit Failover]
Click on this button to modify your Custom Failover Parameters.
You will obtain : Custom Failover Parameters.

Button [ESX Details]
Click on this button to display ESX Details.
You will obtain the ESX Details Window.

Button [VM Details]
Click on this button to display VM Details for the selected VM.
You will obtain the VM Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the ESX Details.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
