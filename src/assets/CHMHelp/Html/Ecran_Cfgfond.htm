Screen Colors

You can modify the following items using this function :
  • The color of the screens
  • The buttons style
  • The color of the Tabs

[Themes] List

This list allows you to quickly set the colors of the various elements according to a particular style. Each selection changes the appearance of the configuration screen to verify the result.

You can save a personal Theme. Just click on the button [Save] and name the Theme (Input a Value). Each new Theme is added to the list.

You can also :
- Rename a Theme by clicking on the [Rename] button.
- Delete a Theme by clicking on the [Delete] button.

Note :
The Theme "Current" at the beginning of the list allows you to display current configuration.


Screen Colors

You can define the colors used in the screens. To do this, first check the case Personnal Colors.
Then you can select the colors :

Button [Background Color]
Click on this button to select a Background color and its Shade. You will obtain the Color Selection Window.

Gradient List
Select on this list the type of background : solid color or gradient (1 to 8).

Button [Text Color]
Click on this button to select the color of texts.

Button [Titles Color]
Click on this button to select the color of the titles at the top of the frames.

Button [Set ...]
Click on this button to define the colors of the input fields and selection lists.
You will obtain the Color Selection Window.

Buttons Style
This list allows you to select the appearance of the buttons :
  • Silver
  • Olive
  • Classic
  • Office
  • SACR
  • Windows 10
Tabs Color
This list lets you choose the color of the Tabs.

Button [Default Values]
Click on theis button to define default values.

Click the [Ok] button to apply your choices.
