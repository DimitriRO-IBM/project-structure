Cluster Overview by VCenter

This window displays the list of the Customer's Virtual Centers, Clusters and ESX.
You can select all the Virtual Centers or a specific Virtual Center using the list "Select the VC".


Commands :
Button [Reload from SRM]
Click on this button to reload the Data from SRM for the selected Customer.

Button [Display URL]
Click on this button to show SOAP URL in a single line.

Button [Test Query]
Click on this button to launch the SOAP Query in the Web Browser.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of the VMs. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
