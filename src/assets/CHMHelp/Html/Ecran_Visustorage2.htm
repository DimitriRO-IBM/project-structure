SOD Provisioning

This window is used to define SOD Provisioning values for the selected Customer.

How to Add a new Line
  • Step 1 : Click the button [Add a Line]

  • Step 2 : Select Provisioning Values :
    - SOD Customer
    - Infrastructure
    - Performance
    - Building
    - Space to Add (in GB)

  • Step 4 : Click the button [Save the Line]

How to Update a Line
  • Step 1 : Select the line on the Table
  • Step 2 : Select Provisioning Values
  • Step 3 : Click the button [Save the Line]

How to Delete a Line
  • Step 1 : Select the line on the Table
  • Step 2 : Click the button [Delete the Line]

How to Copy a Month
You can define SOD Provisioning Values for the selected Month using Provisioning Values used for an other Month.
Click the button [Copy a Month] to copy a Month.


Click on [Ok] to return to the previous window.
