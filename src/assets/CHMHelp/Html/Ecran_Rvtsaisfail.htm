RVTools - Custom Failover Parameters

This window is used to define your Custom Failover Parameters.

You have to select 2 Parameters :
  • The Failover Type :
    - Number of ESX
    - Percentage

  • The Failover Value :
    - 1 ESX
    - 2 ESX
    - 3 ESX
    - 4 ESX

    Or

    - 10 %
    - 15 %
    - 20 %
    - 25 %
    - 30 %
    - 40 %
    - 50 %

Click on [Ok] to update your Custom Failover Parameters.
