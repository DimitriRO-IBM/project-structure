Checking IBM LAN Connection

If SACR cannot connect with IBM Network, you have to verify the following items :

1 - Check SYMANTEC Firewall
  • Open Symantec Interface :



  • Click on the Network Threat Protection Options :



  • Select "View Application Setting ..." :



  • Find the SACR Application on the list, and click on it with the right button.
    Select the Allow option :



  • Click the [OK] Button.

  • Restart SACR to verify.
