RVTools - VM Details

This window displays all the VM Parameters.

What is a Virtual Machine (VM) ?

A Virtual Machine (VM) is a software that, like a physical computer,
runs on Operating System and applications. An Operating System
installed on a Virtual Machine is called "guest Operating System".

Because every Virtual Machine is an isolated computing environment, you can use Virtual Machines as desktop or workstation environments, as testing environments or to consolidate server applications.

In VCenter Server, Virtual Machines run on ESX or Clusters. The same ESX can run many Virtual Machines.

See also : VCenter Architecture
