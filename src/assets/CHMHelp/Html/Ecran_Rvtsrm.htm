RVTools / SRM Cross Checking

This window will help you to compare RVTools Data with SRM Data.
Cross checking is done in 5 steps :

Step 1 - Select the RVTools Customer, VCEnters to be compare and the Date.

Step 2 - Select the SRM Customer with the two lists "Enterprise" and "Available Customers".
Select also the Month of SRM Data.

Step 3 - Click on the button [Validate your Selections]
SACR will read RVTools Data and SRM Data with selected parameters.

Step 4 - SRM VCenters Selection
You have to select (check) SRM Vcenters to be cross checked with RVTools.
Each time you select (check) a SRM VCenter, the program will update the number of concerned Clusters, ESX, VMs Linux and VMs Windows.

Buttons [View]
Click on these little buttons in order to display various lists :

- Lists of Clusters (in RVTools and SRM)
- Lists of ESX
- Lists of VM Linux
- Lists of VM Windows

Note :
RVTools VMs are "Power ON" VMs, SRM VMs are "Active" VMs.

Step 5 - Click on the button [Submit Cross Checking]
The program will display 2 tables :
  • Table 1 - List of Items Present in RVTools but NOT Present in SRM
  • Table 2 - List of Items Present in SRM but NOT Present in RVTools

Commands :

Button [View HTML]
Click on this button to generate a page on HTML format containing the Cross Checking Report.
The page will be displayed using the selected HTML Browser.

Buttons [Copy Table 1] and [Copy Table 2]
Click on these buttons to copy the displayed table 1 or 2 on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
