2 - Export Directory

CSV Export Files can be stored on a specific directory.

Button [Directory Selection ...]
Click on this button to select a specific directory. You will obtain the standard Windows Directory selector.

Button [Default Directory]
Click on this button to select default directory for the Exported CSV Files :

C:\Capacity Reporting\ExportCSV


3 - Commands

Click on this button to generate Export CSV Files.
