Global Inventory

1 - Tab [Count by OS]

This Tab displays the number of Collected Assets of the selected Customer by OS (AIX, Linux, Windows ...).

Button [Sub Client IDs]
By default all "Sub Client ID1" and "Sub Client ID2" are selected. But you can select individualy specific "Sub Client ID1" and "Sub Client ID2" using this button.

Save this Selection
Check this option if you want to save the Sub Clients IDs selection.

You can also select or deselect each OS on the list.

Button [Select All]
Click on this button to select all the OS on the list.

Button [Deselect All]
Click on this button to deselect all the OS on the list.

Button [Reload from SRM]
Click on this button to reload the Global Inventory from SRM :
  • Reload with the Selected Month Date
  • Reload with Latest data Available

Note : The Total of Assets doesn't count the "Clusters" line because they are not physical Assets.

Button [Display Queries]
Click on this button to show SOAP URLs.
You will obtain the SRM Queries Window.


2 - Tab [End of Support]

This Tab displays a table and some graphs.
  • The table shows, for every OS, the version number, the date of the End of Support and the number of Assets (Servers, LPARs or VMs). SACR counts the number of Assets with the OS version "Ok", "To be Upgraded", "Not Supported" and "Not Applicable".
    The last line of the table displays the different totals.

  • The first graph is the "TOTAL OF ASSETS" graph.
    The following graphs are done for every OS.

End of Support Date Selection
You can display End of Support Counts using one of the 2 following dates :
  • Full Support Date
  • End Of Life Support Date
The 2 dates are displayed on the table in this way :
Full Support Date | End Of Life Date

Exemple :
01-2015 | 01-2020

Button [Options]
Click on this button to select the OS Configuration File to be used.
You will obtain the End of Support Options Window.

Button [End of Support Forecast]
Click on this button to display the End of Support Forecast Window.


3 - Tab [Obsolete SRM Agents]

The table shows, for every OS, the available version of each SRM Agent and the date of availability.
SACR counts the number of Assets with the SRM Agent "Ok" and "To be Upgraded".
The last line displays the number of Assets with undefined SRM Agent version.


4 - Tab [Up Times Days]

The table shows a list of Up Time periods, for Unix and Windows Assets.


5 - Tab [Assets Inventory]

This Tab displays the list of Collected Assets of the selected Customer.
You can sort the list clicking on the head of the columns.


Search a Server
To search a Server name on the list, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Server line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Servers.

You can select Assets to be listed :
- Collected Assets
- Collected and Not Collected Assets
- Not Collected Assets

Button [Columns Selection]
You can display or hide every column of the Collected Assets List.
Click on this button the select the columns to be displayed.

Button [Browse ...]
You can compare the contains of an external CSV file with the Global Inventory provided by SRM. Just click on this button to select and open an external CSV file. This file must have one column with the Hostnames. Then, SACR will compare this file with SRM information. Results are displayed in two lists :
  • Servers on File NOT in SRM
  • Servers on SRM NOT in File

Button [Paste]
You can directly paste a list of Hostnames using this button. SACR will compare this list with SRM information in the same way than described before.

Buttons [Copy]
Click on this buttons to copy data from the selected list into the Windows Clipboard. The program will ask you to select first the Field Separator. You will obtain the Field Separator Window.


6 - Tab [Collect Quality]

This Tab displays the SRM Data Collect Quality (in %) for each Asset.
You can sort the list clicking on the head of the columns.


Account View
Select this option to display a summary of Data Collect Quality for all the Boxes (Pseries and VCenters).
You can double click on a line to get the list of Assets of the PSeries or the VCenter.

Box View
Select this option to display details for a specfic Box (Pseries or VCenter).

- You can then select a Box using the dropdown list.
- You can also make a filter introducing a % value in the input field.

Collect Quality Formula.


Commands :

Buttons [Details]
You can select / deselect one or more lines on different tables using the [Ctrl]+Clic command. Then, you can display the list of Assets concerned by the selection by pressing the [Enter] key or using the button [Details].
You will obtain the Details Window.
Note : A double click on a line will open the Details Windows for the selected line.

Button [Audit Report]
Click on this button to generate a page on HTML format containing a complete Audit Report.
The page will be displayed using the selected HTML Browser.
The Audit Report displays the following items :

1 - Count by OS
2 - End of Support (Collected Assets only)
3 - List of Not Collected Assets in SRM
4 - Up Times Report
5 - Obsolete SRM Agents


Button [View HTML]
Click on this button to generate a page on HTML format containing the list of Assets. The page will be displayed using the selected HTML Browser.

Button [OS Report]
Click on this button to display the Global OS Report for the selected Customer.

Button [Copy Table]
Click on this button to copy the displayed table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [Export CSV]
Click on this button to export Global Inventory lists of Assets to a CSV file.
You will obtain Export to CSV Files window.

Button [Quit]
Click on this button to return to the previous screen.
