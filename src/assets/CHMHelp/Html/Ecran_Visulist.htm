List of Servers

This window displays the list of the Servers of the selected Customer.
You can sort the list clicking on the head of the columns.

Status Columns
The list displays status for some metrics in Green, Yellow or Red :
P : Processor
M : Memory
D : Disk
V : Volume Group
F : Filesystem

Commands :

Search a Server
To search a Server name on the list, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Server line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Servers.

Button [View Details]
Click on this button to open the View Details window for the selected Server on the list. You obtain the same result with a double click or pressing the [Enter] key.

Button [Monthly View]
Click on this button to display CPU Daily data and CPU Hourly data for the selected server. You will open CPU Monthly View.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of the Servers. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
