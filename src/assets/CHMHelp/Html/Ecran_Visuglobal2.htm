Enf of Support Forecast

Parameters :

Forecast Incrementation
Use this list to select the Forecast Incrementation value :
  • 3 Months
  • 6 Months
  • 9 Months
  • 12 Months

Collapsed View.
Check this case to display the Collapsed View of the Forecasts.

Values in %.
Check this case to display Forecast Values in a percentage of the number of Assets.


Commands :

Button [Copy Table]
Click on this button to copy the displayed table on the Windows Clipboard. You will first obtain Copy Table window in order to select the copy format. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
