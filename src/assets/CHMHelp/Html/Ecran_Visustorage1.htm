Storage Reports

Information :

In SACR the "Storage Reports" function displays data for BROD, SOD and SAN, for the French customers who use the shared infrastructure and some customers who have a dedicated infrastructure.
The data comes from different data sources (GBRS, SOD, ..), and is concatenated once a day on the SACR server.
When you use SACR the concatenated data is downloaded from the SACR server and stored locally on your workstation.
The same day, if you use SACR again the data is retrieved from your workstation, except if you force the download again.
At the beginning of a month, the BROD Occupancy data of the previous month is available only on the 2nd day, because the TSM batch jobs that produce the data last several days, and the BROD Transferred data is available only on the 5th day.


This window displays Reports for the following Storage Domains :
  • SOD Reports
  • BROD Occupancy Reports
  • BROD Transferred Reports
  • VM Occupancy Reports
  • VM Transferred Reports
  • SAN Reports (Ports Count)
  • DSU Reports

By default the program selects all the Customers defined for the selected Storage Domain. But you can select a specific Storage Customer using the selection list in the top of the window.

Commands :

Month Selection
You can select with this list months to be displayed. You can use also the following buttons :

- [Highlighted]
- [Last 3 Months]
- [Last 6 Months]
- [Last 9 Months]
- [All the Months]

Display / Infra / Metric
You can select the Data to be displayed on the graphic. The contents of this list depends of the Storage Report (SOD, BROD, VM or SAN).

For the BROD Reports


Split Type
For BROD Occupancy Reports, you can display measures splitted in 6 columns :

- Avg ONSITE
- Avg OFFSITE
- Avg EVAULT
- Max ONSITE
- Max OFFSITE
- Max EVAULT

Details by BROD Customer
For BROD Transferred Reports, you can display the sum on the values for all the BROD Customers or display values for each Customer.

Values in TB
Select this option to display values in TB (Terabytes).

Daily Values
For BROD Transferred and VM Transferred reports you can display daily values selecting this option.

Average Values
For VM Occupancy reports you can display Monthly Average Values or Monthly Sums selecting this option.

Stacked Bars
Select this option to display Stacked Bars on the graph. Bars are displayed one in front the other instead of one beside the other.

Show Forecast
Select this option to display Forecast Values.

Forecast Months
You can select with this list the number of Forecast months to be displayed on the graph.

Button [Columns Selection]
You can display or hide every column of the table.
Click on this button the select the columns to be displayed.

Button [Reload Data]
Click on this button to reload Storage Data from SACR Server.

Button [List of Nodes]
For BROD Reports, click on this button to display the list of Nodes.
You will obtain the List of Nodes window.

Button [List of LUNID]
For SOD Reports, click on this button to display the list of LUNIDs.
You will obtain the List of LUNIDs window.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format. You can select items to be displayed.
The page will be displayed using the selected HTML Browser.

Button [Export to CSV]
Click on this button to export data to a CSV File.

Button [Provisioning]
For SOD Reports, you can define Provisioning values with this button.
You will obtain the SOD Provisioning window.

Button [Quit]
Click on this button to return to the previous screen.
