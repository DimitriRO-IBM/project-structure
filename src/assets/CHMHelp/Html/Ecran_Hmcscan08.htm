LPAR Distribution

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The HMC Name
  • The Date
Each time you select a new Customer, SACR will update automatically the "HMC" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the HMC list contains more than 2 HMC, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom HMC Selection.

LPAR Status Selection
Select with this list the Status of LPARs to be displayed :
  • All LPARs (ON and OFF)
  • Only Running LPARs

List of Boxes
When you click on a Box in the list, LPARs Distribution of this Box is displayed, with the 3 graphs "VP", "EC" and "Memory" Distributions.
If you double-click on a Box, or if you press the [Enter] key, you will open the Box Details Window.

LPARS Distribution
Metrics are divided in 8 Classes. The number of LPARs for each Class is calculated and plotted in a graph.
If you click on a Class, the corresponding LPARs are displayed below.

List of LPARs
You can display LPAR details with a double-click on a LPAR or pressing the [Enter] key.


Commands :

Button [Box Details]
Click on this button to view details of the selected Box.
You will obtain the Box Details Window.

Button [LPAR Details]
Click on this button to view details of the selected LPAR.
You will obtain the LPAR Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format.
The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the list of LPARS on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
