Storage Reports - List of LUNID / List of Nodes

This window displays :
  • The list of LUNID for SOD Reports
  • The list of Nodes for BROD Occupancy Reports
  • The list of Nodes for BROD Transferred Reports

Commands :

Month Selection
You can select with this list months to be displayed. You can use also the following buttons :
- [Highlighted]
- [Last 3 Months]
- [Last 6 Months]
- [Last 9 Months]
- [All the Months]

Search
To search a TSM Server or a Node name on the list, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Node line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Nodes.

Button [Columns Selection]
You can display or hide every column of the table.
Click on this button the select the columns to be displayed.

Button [Reload Data]
Click on this button to reload Storage Data from SACR Server.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format. You can select items to be displayed.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
