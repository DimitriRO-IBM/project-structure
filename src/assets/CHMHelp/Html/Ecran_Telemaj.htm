Download the Last Version

You can download at any time the last version of the program. The installed version on your computer is displayed on the top of the window.


Commands :

Button [Version History]
Click on this button to view the History of the different versions of the program.

Button [Download the Last Version]
Click on this button to download the last version of the program.

Check the new Versions
Tick this box if you want to check for a new version at every program startup.
