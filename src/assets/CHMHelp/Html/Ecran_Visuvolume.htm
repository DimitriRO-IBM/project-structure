Volume Groups

This window displays the list of Volume Groups (or Logical Units) for the selected Server.
The top of the window gives general information :

- Customer Name
- Server Name
- Subclient ID 1
- Subclient ID 2
- Type ("Server" or "LPAR")
- Operating System

Commands :

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. You will first obtain Copy Table window in order to select the copy format. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graphic. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
