Global Report

This window is used to define what will be the Reports to be included on the Global Report.

1 - General Information

The program displays on the top of the window the Customer Name, the selected Month, the Operating System and the Display (All Servers or Top 20).

On the bottom of the screen, the program displays the Files Directory and the Archives Directory. These Directories can be defined using the command "Menu-Setup-Options", frame "Reports Directory".

3 - Commands

Button [Select All]
Click this button to select all the Reports.

Button [Deselect All]
Click this button to deselect all the Reports.

Button [Create Report]
Click on this button to generate the Global Report.
You can create a Report using 4 methods :

  • Word, Automatically
    The Word Report will be generated automatically using Word commands.

  • Word, Manually
    In this case, Word Report will be generated step by step by yourself. You will obtain Word Report Creation window.

  • HTML Only
    In this case only the HTML Report File will be generated.

Button [Create Archive]
You can create a ZIP file containing all the files used to display the Global Report by clicking on this button. A different Archive file is generated for each type of display ("All Servers" or "Top 20").


Button [View Word Report]
Click on this button to open the Word Global Report in memory.


Button [View HTML Report]
Click on this button to open the HTML File of the Report.
