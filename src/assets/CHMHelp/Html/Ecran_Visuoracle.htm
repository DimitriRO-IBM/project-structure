Oracle DB Sizes

This window displays, for the selected Customer :
  • A Table containing the list of all Oracle Databases for the selected Customer
  • 2 Graphs showing, for the selected database on the table, the values for the last 12 months :

    Graph 1 :
    - Verticals bars show the Allocated Size (in GB).
    - Line gives variations of the DB Used Percentage.

    Graph 2 :
    - Verticals bars show the count of User Names logged.
    - Line gives variations of the "Buffer Hit Ratio". This value is the Percentage of time where data is found present in memory.


Commands :

You can select the contents of the list :
  • Display the Selected Month
  • Display the last 12 Months

Display only Red Status
Select this option if you want to display only lines with Red Status.

Button [Reload from SRM]
Click on this button to reload Oracle information from SRM.

Button [SOAP Queries]
Click on this button to launch SOAP Queries in the Web Browser.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Buttons [Copy Graph 1] and [Copy Graph 2]
Click on these buttons to copy the selected graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.
