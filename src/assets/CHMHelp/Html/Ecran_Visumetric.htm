View Metrics

This window displays 3 Tabs :
  • A Table containing the list of the Servers and the associated metrics
  • A Graph showing the values of the selectd Metric (CPU, Memory or Disk)
  • A list of Missed Data or Servers (all metrics)

Commands :

Search a Server
To search a Server name on the list, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Server line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Servers.

Thresholds
You can modify the values of the different Thresholds. Then you can :

- Click the [Update Resuts] button to update the Table and the Graphic using your own Thresholds.

- Click the [Defaults] button to update the Table and the Graphic using the default values stored by the program. Defaults Thresholds values are saved using the command "Menu-Setup-Thresholds".
See also Setup Thresholds.

Sort by Default
You can select the sort by default of the list of Servers :
- By Value
- By Status (colors)
- By Subclient ID 1
- By Subclient ID 2
- By Server Name

Then you can click on the name of each column to sort values. A second click will sort values in a descending order.

Type of Report
The program allows you to display two kinds of Reports (on CPU and Memory) :

Highest Values
Lowest Values

Display Forecasts
You can then select the months to be Forecasted :
- None
- 1 and 3 Months
- 3 and 6 Months
- 6 and 12 Months

Box Names
You can hide or display the column "Box Name" using this option.

Button [View Details]
Click on this button to open the View Details window for the selected Server on the list. You obtain the same result with a double click or pressing the [Enter] key.

Button [Monthly View]
Click on this button to display CPU Daily data and CPU Hourly data for the selected server. You will open CPU Monthly View.

Button [Forecast View]
Click on this button to display Forecast Values for the selected server. You will open Forecast View.

Button [Volume Groups]
Click on this button to display the list of Volume Groups of for the selected server. You will open Volume Groups.

Button [Filesystems]
Click on this button to display the list of Filesystems of for the selected server. You will open Filesystems.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. You will first obtain Copy Table window in order to select the copy format. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.
