RVTools - Datastore Details

This window displays all the Datastore Parameters.

What is a Datastore ?

Datastores in vSphere are storage containers for files. They could be located
on a local server hard drive or across the network on a SAN.
Datastores hide the specifics of each storage device and provide a uniform model
for storing virtual machine files.

See also : VCenter Architecture


Commands :

Button [ESX Details]
Click on this button to display ESX Details.
You will obtain the ESX Details Window.

Button [VM Details]
Click on this button to display VM Details for the selected VM.
You will obtain the VM Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the ESX Details.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
