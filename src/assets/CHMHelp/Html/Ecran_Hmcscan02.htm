HMCScan - Excel File Import

Import of an HMCScan Excel File is done in 4 steps :

Step 1 - Type, or select with the list, the Customer Name
Type or select the Customer Name with the combo list.

Step 2 - Excel File Selection
Click on the button [Browse ...] to select and open an HMCScan Excel File.
Imported Data :
  • HMC Manager Info
  • List of Boxes
  • LPAR Summary
  • LPAR Profiles
  • Physical Slots
  • IO Slot Children
  • Virtual SCSI
  • VIOS Disks
  • Fiber Channels

Button [Modify Tables]
If necessary, you can modify now the name of the Customer using these button.
Just give the name of the Customer on the combo box, and click on the |Modify Tables] button.

Step 3 - Define Boxes Locations
HMCScan Excel Files do not have the "Box Location" information. So you have to input this information manually using the button [Define Boxes Locations] (optional).
You will obtain the Boxes Locations Window.

Step 4 - Save Data in HMCScan Database
Click on the button [Save Data in Database] to finish import.

Button [Quit]
Click on this button to return to the previous screen.
