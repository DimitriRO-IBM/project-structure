1 - Menu Bar

These are the main commands :
  • File
    • Calendar
    • Return to the Main Windows

  • Setup
    • Options
    • Graphics


2 - File Selection

You have to select first :
  • The Customer
  • The Metric (CPU, Memory, I/O)
  • The Data File to be loaded

3 - Data Selection

Select in this panel the following filtering parameters :
  • The Year
  • The Month
  • The Week
  • The Day of the Week
  • The Hour
  • The LPAR / System
  • The Mesure to be processed
Select the Measure on the list an the type of calculation (Average or Sum).

Select then the Display :

Button [Houtly View]

Button [Daily View]

Button [Days and Hours]


4 - Commands

Button [Zoom]
Click on this button to increase the size of the Data table.

Button [Settings]
Click on this button to set the graphics options. You will obtain the Graphics Setup Window.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the Main Window.
