  • Download Data from the SACR Server

    Display Options :
    - Infrastructure View
    - Boxes View
    - Volume Groups
    - Filesystems

  • Month -4
  • Month -3

The first Tab "Last 12 Months" displays the Servers for the selected Customer for the 12 last months. The other Tabs display data for the previous months.

View Details
You can obtain Metrics for a specific Server just by double-click on it (or pressing the [Enter] key).
You will obtain the View Details Window.

Button [Display SOAP]
This button allows you to display the SRM SOAP Queries used to import data. Queries are dynamically set in function of selected Customer, Month, OS and Shift. You will obtain the SOAP Queries window.
SOAP : Simple Object Access Protocol

Button [SOAP Availability]
Click on this button to check SRM SOAP availability. A simple SOAP Query will be used to check SOAP.

Button [Hide Data Details]
Click on this button to hide Data Details.

Button [Copy]
Click on this button to copy data from the selected Tab into the Windows Clipboard. The program will ask you to select first what you want to copy :
- 1 : The Selected Line
- 2 : All the Table

If you want to copy the entire Table, the program will then ask you to select the Field Separator. You will obtain the Field Separator Window.

Button [Show Empty] / [Hide Empty]
By default, empty columns are hidden (columns without any data). Click on this button to display columns in all the tables. Then click on [Hide Empty] to hide again empty columns.

Button [Search and Select]

Search a Server
To search a Server name on the list, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Server line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Servers.
