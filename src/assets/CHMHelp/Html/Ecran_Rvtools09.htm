RVTools - Disaster Recovery Plan (DRP) Simulation

Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.

Scenario Selection
The program offers the possibility to define 3 different Disaster Recovery Scenarios.
Select here the Scenario you want to use :
  • Scenario 1
  • Scenario 2
  • Scenario 3
Buttons [Previous Step] and [Next Step]
The DRP Simulation is done in 3 steps. Use these buttons to move to the next step or return to the previous one.


Step 1 - Pairing Clusters

Button [Pairing]
You have to define, for each Protected Cluster, the Recovery Cluster wich is going to be used for the Disaster Recovery Plan (DRP) Simulation.
Select first the Protected Cluster on the left list, then select the Recovery Cluster on the right list, and click the button [Pairing] to create the Recovery Cluster Association.
You can also filter Clusters by ESX Sites if you want.

The left list will display, for each Protected Cluster, the Associated Recovery Cluster in this way :
Protected Cluster --> Recovery Cluster

Button [Clear Pairing]
You can delete a Cluster Pairing using this button. A confirmation message will ask you to confirm this action.

Double-click on a line to see more Cluster Details.

Note :
You have to select the same Date for Multiple VCenter DRP.

Selection of Clusters to be Protected
You have to select (check / uncheck) in this list the Clusters to be Protected during the Disaster Recovery Scenario you select in the top of the window.
Selections are automatically saved.


Step 2 - VM Selection

In this step you will select VMs to be protected in a more precise way.
You can select VMs using these two lists :
  • List 1 : Selection of VMs by Datastore
  • List 2 : Selection of VMs ON one by one
Just check / uncheck VMs to be added from the Protected Clusters to the Recovery Clusters.
If you check / uncheck a Datastore, all VMs of this Datastore will be checked / unchecked.

Button [Select All VMs]
Click on this button to select all VMS to be added from the Protected Clusters to the Recovery Clusters.
Selections are automatically saved.


Step 3 - Disaster Recovery Plan Simulation

The window displays 2 tables containing data of the selected Recovery Clusters.
For each Recovery Cluster, tables give these information :

Table 1
01 : ESX Count
02 : Number of VM ON Before DRP
03 : Number of Real Cores
04 : Total Virtual CPU (vCPU) Before DRP
05 : Effective Memory (GB)
06 : Total vRAM (in GB) Before DRP
07 : Number of VM ON After DRP
08 : Total Virtual CPU (vCPU) After DRP
09 : Total vRAM (in GB) After DRP

Table 2
09 : Average vCPU Slot
10 : Average vRAM Slot
11 : Available Slots
12 : Selection Column [•]
13 : DRP Ratio 1 : vCPU / Cores
14 : DRP Ratio 1 : Applied Value
15 : DRP Ratio 2 : vRAM / Effective Mem
16 : DRP Ratio 2 : Applied Value

Commands :

Button [Copy Tables]
Click on this button to copy the displayed tables on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Cluster Selection
You can select a Cluster in the table 2 using one of these methods :
  • Click on the column [•]
  • Press the [Space] bar

Button [Select All]
Use this button to select all the Clusters in the table.

Button [Deselect All]
Use this button to deselect all the Clusters in the table.

Button [Update Applied Ratios]
Click on this button to define your Personnal DRP Ratios 1 and 2.
You will obtain : CPU and Memory Applied Ratios.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of Items. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
