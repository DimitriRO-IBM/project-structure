VCenter Allocated Resources

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.

View Selection
You can select the Allocated Resources View :
  • By Clusters
  • By ESX
For the Clusters view, you can also select one on this 3 modes :
Nominal Mode

Failover or Maintenance Mode

Custom Failover Mode

When you willingly place an ESX host in maintenance to service it or when the host fail unexpectedly for some reasons the Virtual machines that are running on this host will need to be migrated to another one (either manually or automatically by DRS) or shut down.

2 - Allocated Resources

The window displays 2 tables containing data of the selected VCenters.
For each Cluster, tables give these information in Nominal Mode :

Table 1
01 : ESX Count
02 : Number of VM ON
03 : Number of VM OFF
04 : Number of Real Cores
05 : Total Virtual CPU (vCPU)
06 : Available Virtual CPU (vCPU)
07 : Effective Memory (GB)
08 : Total Virtual Memory (vRAM) (GB)
09 : Available Virtual Memory (vRAM)

Table 2
10 : Average vCPU Slot
11 : Average vRAM Slot
12 : Available Slots
13 : Selection Column [•]
14 : Ratio 1 : vCPU / Cores
15 : Ratio 1 : Applied Value
16 : Ratio 2 : vRAM / Effective Mem
17 : Ratio 2 : Applied Value

Commands :

Cluster Selection
You can select a Cluster in the table 2 using one of these methods :
  • Click on the column [•]
  • Press the [Space] bar

Button [Select All]
Use this button to select all the Clusters in the table.

Button [Deselect All]
Use this button to deselect all the Clusters in the table.

Button [Update Applied Ratios]
Click on this button to define your Personnal Ratios 1 and 2.
You will obtain : CPU and Memory Applied Ratios.

Button [Recommendations for Selected Cluster]
Click on this button to display an complementary information message.

Button [Failover Info]
This button displays how are calculated values of CPU and Memory on Failover or Maintenance modes.

Button [Edit Failover]
Click on this button to modify your Custom Failover Parameters.
You will obtain : Custom Failover Parameters.

Button [Cluster Details]
Click on this button to view details of the selected Cluster.
You will obtain the Cluster Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of Items. The page will be displayed using the selected HTML Browser.

Button [Copy Tables]
Click on this button to copy the displayed tables on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
