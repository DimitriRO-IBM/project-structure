HMCScan - LPAR Details

This window displays all the LPAR Parameters.

Commands :

Button [Box Details]
Click on this button to display BOX Details.
You will obtain the Box Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the VM Details.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
