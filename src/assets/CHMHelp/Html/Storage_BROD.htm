Backup and Restore Types
  • ONSITE
    Backup Data is located in the TSM Robot (Primary Data).

  • OFFSITE
    ONSITE Data is copied on a Tape Set. These bands are then removed from the TSM Robot to be
    sent by truck to another site (Aix en Provence, approximately 200 kms).
    To recover the OFFSITE Data, you have to return back the Tapes by car, and re-insert them into the robot.
    This operation can last several hours.

  • EVAULT
    ONSITE Data is copied using the Network to another TSM Robot located in Grabels.
    No manipulation of Tapes is needed. The EVAULT Data is available almost instantly.
