RVTools - ESX Details

This window displays all the ESX Parameters.

What is an ESX (also named Host) ?

An ESX (or Host) is a computer that uses virtualization software, such as ESX or ESXi,
to run Virtual Machines (VMs).
ESX provides :
- The CPU and Memory resources that Virtual Machines use
- Give to the VMs access to Storage
- Give to the VMs Network connectivity

See also : VCenter Architecture


Commands :

Button [Cluster Details]
Click on this button to display Cluster Details.
You will obtain the Cluster Details Window.

Button [VM Details]
Click on this button to display VM Details for the selected VM.
You will obtain the VM Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the ESX Details.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
