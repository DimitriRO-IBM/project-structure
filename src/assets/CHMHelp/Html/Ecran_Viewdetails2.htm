Monthly View / 15 Minutes View

This window displays Weekly Data, Daily Data, Hourly Data and 15 Minutes Data for the selected Server and the selected Metric.
The top of the window gives general information :

- The Month to be displayed
- The Customer Name
- The Server Name
- The Type ("Server" or "LPAR")
- The corresponding CPU Thresholds

View Selection

Click on the selected Tab [Weekly Details], [Daily Details] or [Hourly Details].

[Hourly Details] and 15 Minutes View
You can zoom the graph using the Zoom option.

Display
For AIX (Lpars and Servers) and LINUX (Servers) Memory Details, you can select the Metric to be displayed on the graph :

AIX :
- Memory Swap Pct Used
- Memory Available for Applications (Mo)
- Memory Available for Applications (%)

LINUX Server :
- Memory Used Percent
- Memory Available for Applications (Mo)
- Memory Available for Applications (%)

Calculation of [Memory Available for Applications (Mo)]

These are the formulas used to calculate this Metric :

AIX :
[mem_pages_avail_applications] = MAX((free pages - minfree),0) + MAX((file pages - minperm),0)
[Memory Available for Applications (Mo)] = ( [mem_pages_avail_applications] x 4096 ) / (1024 x 1024)

LINUX Server :
[memory_effective_used_percent] = (memory free + memory buffer + memory cache)/(installed memory)
[Memory Available for Applications (Mo)] = [memory_size] - ( [memory_effective_used_percent] x [memory_size] / 100 )

Display Forecast

Button [SOAP Queries]
When you click on this buuton you will get the following commands :
  • Display URLs
    Select this command to show SOAP URLs in a single line.

  • Test Weekly Query
    Select this command to launch the Weekly SOAP Query in the Web Browser.

  • Test Daily Query
    Select this command to launch the Daily SOAP Query in the Web Browser.

  • Test Hourly Query
    Select this command to launch the Hourly SOAP Query in the Web Browser.
  • Test 15 Minutes Query
    Select this command to launch the 15 Minutes SOAP Query in the Web Browser.

Button [Vew VM Details]
This button is available only for VMWARE ESX Reports.
You will obtain the ESX - Daily VM Details Window.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph of the selected Tab on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing Tables for the Daily View and the Hourly View. The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
