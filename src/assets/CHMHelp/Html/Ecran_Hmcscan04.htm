HMCScan Details

This window displays the list of Assets concerned by the selected criteria.
You can sort the list clicking on the head of the columns.

Searching
To search an Item, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Item line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Items.


Commands :

Button [Details]
Click on this button to obtain more information on the selected Asset :
  • Box Details
  • LPAR Details

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of the Servers. The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Line]
Click on this button to copy the selected line on the Windows Clipboard.

Button [View SQL]
Click on this button to display the SQL Query used to load data.

Button [Quit]
Click on this button to return to the previous screen.
