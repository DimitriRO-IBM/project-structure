Graphics Setup

This window is designed to customize the appearance of graphics. The options are as follows :
  • Background Color
  • Texts Color
  • Font Name
  • Font Size
  • Curves Thickness
  • Gradient of Bars
  • Horizontal Grid (Yes / No)
  • Vertical Grid (Yes / No)
  • Solid Grid (Yes / No)
  • Vertical Bands (Yes / No)
  • Double Graduation (Yes / No)
Colors
To select a color, click on the corresponding button. You will obtain the screen Color Selection.

List [Theme]
This list allows you to quickly set the colors of the various elements based on a particular theme (SACR, Marine, Desert ...).


Click on the button [Ok] to save Graphics Parameters, or [Cancel] to quit without changes.
