Profile Selection

You can define personnal Sub Clients IDs Profiles for each Customer and OS. A Profile is a list of "Sub Clients ID 1" and a list of "Sub Clients ID 2". You can then store these 2 lists on a Profile. Each Customer can have one or more Profiles.

The selected Customer and OS are displayed on the top of the window.

1 - Profile Selection

This list contains Profiles created for the selected Customer and OS. You can select directly a Profile, or give a name if you want to create a new Profile.


2 - "Sub Clients ID 1" and "Sub Clients ID 2" Selection

The two lists display "Sub Clients ID 1" and "Sub Clients ID 2" available for the Customer. Select or deselect items using checks boxes on the left of each list.

Button [Selection by Criteria]
Click on this button if you want to select "Sub Clients ID 2" using a search criteria. You will first obtain the Value Selection window. The program will select all the "Sub Clients ID 2" containing this criteria.

Button [[X]]
Click on this button to select all the "Sub Clients IDs" on the list.

Button [[ ]]
Click on this button to deselect all the "Sub Clients IDs" on the list.


3 - Save or Delete a Profile

Button [Save]
Click on this button to save the Profile.

Button [Delete]
Click on this button to delete the selected Profile.

Button [Ok]
Click on this button to validate the "Sub Clients ID 2" selection and return to the Main Window.
