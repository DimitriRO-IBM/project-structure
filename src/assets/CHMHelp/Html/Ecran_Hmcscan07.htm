Profiles Cross Checking


This window displays Profile Differences between Active Profiles (P) and Running LPARs (L).

Commands :

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of items.
The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the list on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
