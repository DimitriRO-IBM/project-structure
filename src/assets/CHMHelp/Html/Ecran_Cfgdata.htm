Manage Data Files

This window displays the List of the Customers having Data Files saved on your disk.
By default, Data Files are saved on the ...\Data Directory, in the program's directory.

Button [Open File]
Click on this button to open the selected Data File. Files are .csv format Files.

Button [Open with Norepad]
Click on this button to open the selected Data File with Windows Notepad.

Button [Windows Explorer]
Click on this button to open the selected Data File on Windows Explorer.
