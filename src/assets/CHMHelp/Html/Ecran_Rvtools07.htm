RVTools - Ports and Switches

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The VCenter Name
  • The Date


Each time you select a new Customer, SACR will update automatically the "VCenter" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the VCenter list contains more than 2 VCenters, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom VCenter Selection.

Searching
To search an ESX, type the characters to be found and click the button, or press the [Enter] key.
If the criteria matches, the Item line will be highlighted.
Use [Previous] and [Next] buttons to highlight the others Items.


2 - List of ESX

This list displays ESX for the selected Customer, VCenter and Date. Click on the name of each column to sort it. You can obtain more details for each ESX with a double click on a ESX line.
Each time you click on an ESX, SACR will display two secondary lists :
  • ESX Port Groups
  • ESX Switches

3 - List of Distributed Ports

This list displays Distributed Ports for the selected Customer, VCenter and Date. Click on the name of each column to sort it.


Commands :

Button [ESX Details]
Click on this button to display all the ESX details.
You will obtain the ESX Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format containing the list of Port Groups and the list of Switches for the selected ESX.
The page will be displayed using the selected HTML Browser.

Button [Quit]
Click on this button to return to the previous screen.
