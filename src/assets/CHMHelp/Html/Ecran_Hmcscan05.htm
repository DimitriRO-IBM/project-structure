Allocated / Available Resources

1 - Customer Selection

The top of the window allows you to select the 3 following parameters :
  • The Customer Name
  • The HMC Name
  • The Date
Each time you select a new Customer, SACR will update automatically the "HMC" list and the "Date" list.
You have to click on the [Ok] button to reload new data.

Button [...]
If the HMC list contains more than 2 HMC, you will be able to do a "Custom Selection" using this button.

Button [?]
This button is used to display the current Custom HMC Selection.

Box Thresholds Definition
You can define here 2 Thresholds :
  • Current Available Cores
  • Current Available Memory

Button [Apply]
Click on this button to apply selected Thresholds.

Button [Default]
Click on this button to apply defaults Thresholds :
- Current Available Cores < 5 %
- Current Available Memory < 10 %

Boxes Status Selection
Select with this list the Status of Boxes to be displayed :
  • All Boxes (ON and OFF)
  • Only ON Boxes

List of Boxes
When you click on a Box in the list, the graphs "Cores" and "Memory" are updated with the selected Box information.
If you double-click on a Box, or if you press the [Enter] key, you will open the Box Details Window.

Metric Selection
You can select with this list the values displayed on the graph :
  • Total Number of LPARs
  • Number of LPARs ON
  • Number of LPARs OFF
  • LPARs ON and OFF
  • Total LPARs, ON and OFF
  • Number of Cores
  • Memory Size

Commands :

Button [Box Details]
Click on this button to view details of the selected Box.
You will obtain the Box Details Window.

Button [View HTML]
Click on this button to generate a page on HTML format. You can select items to be displayed.
The page will be displayed using the selected HTML Browser.

Button [Copy Table]
Click on this button to copy the list of Boxes on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the displayed graph on the Windows Clipboard. You will obtain the Graphic Size Selection window.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [Quit]
Click on this button to return to the previous screen.
