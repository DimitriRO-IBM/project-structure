RVTools - CPU and Memory Applied Ratios

This window is used to define your Personnal Ratios for CPU and Memory.

In the left of the window you can see the selected Clusters.
You have to input 2 Ratios :

CPU
Ratio 1 : vCPU / Cores

Memory
Ratio 2 : vMEM / Effective Mem

Click on [Ok] to update your Personnal Ratios.
