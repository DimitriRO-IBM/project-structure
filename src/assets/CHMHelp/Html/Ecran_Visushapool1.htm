Shared Pool

This window displays Shared Pool data for the selected Customer.

View Selection

Select first one of the following options :

1 : Hourly View for the 30 last days

2 : 15 Minutes View for the last 7 days

3 : 15 Minutes View Combined with Hourly View, for the last 7 days
Then you can select :
  • The Box to be displayed
  • The Day to be displayed (or "All Days")

Other Options :

Display Trend Line
Select this option if you want to display a Trend Line on the graph.

Zoom
Select this option to increase the width of the graph.


Commands :

Button [View LPAR Details]
Click on this button to load from SRM the values associated wich every LPAR.
You will obtain the Details by LPAR window.

Button [Update Data from SRM]
Click on this button to update data (Hourly and the 15 Minutes Views) from SRM with the last 30 days.

Button [Display URLs]
Click on this button to show SOAP URLs in a single line.

Button [Copy Table]
Click on this button to copy the table on the Windows Clipboard. The program will ask you to select first the Field Separator.
You will obtain the Field Separator Window. You can then paste the table on another program using [Cntrl+V] shortcut.

Button [Copy Graph]
Click on this button to copy the graph on the Windows Clipboard. You will obtain Graphic Size Selection.
Then you can paste the graph on another program using [Cntrl+V] shortcut.

Button [View HTML]
Click on this button to generate a page on HTML format containing the Table and the Graph. The page will be displayed using the selected HTML Browser.

Graphic Commands
These are the Graphic Commands.

Button [Quit]
Click on this button to return to the previous screen.
