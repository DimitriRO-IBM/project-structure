SOAP Queries

This window shows the SOAP Queries used to download data from SRM.

Commands :

Button [Test Query]
Click on this button to launch the selected Query on the Internet Browser.

Button [Copy Query]
Click on this button to copy the selected Query on the Windows Clipboard.

Button [Copy All Queries]
Click on this button to copy all Queries on the Windows Clipboard.

Button [Display SRM Fields]
Click on this button to show all SRM Fields used in SOAP Queries.

Button [Quit]
Click on this button to return to the previous screen.
